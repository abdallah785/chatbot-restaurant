Recevez vos commandes sur Telegram en 2 minutes

1
Telechargez Telegram
Telegram est une application de messagerie gratuite et rapide. Installez-la sur votre telephone.
🍎 App Store ▶️ Google Play
2
Creez un groupe
Ouvrez Telegram et creez un nouveau groupe :

Nouveau groupe → Nommez-le Commandes [votre resto] → Creer
3
Ajoutez le bot
Dans le groupe, cliquez sur le nom du groupe en haut → Ajouter un membre → Cherchez @Aifood0_bot → Ajoutez-le
4
C'est pret !
Chaque commande arrivera dans ce groupe avec tous les details : plats, modifications, type (sur place / emporter), nom du client.
Astuce : Vous pouvez ajouter toute votre equipe dans le groupe. Tout le monde verra les commandes en temps reel.
